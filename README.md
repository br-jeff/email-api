# Email API
  A simple NodeJS API to send email
## Requirements
  You need install docker and docker compose.
  To run use:
  ```
  $ docker-compose up
  ```

## NodeJS API
 ```
  http://localhost:8080/
 ```


| METHOD | Path | Description|
| -----|------- |:-------------:|
| GET |  /  | Just return a simple Hello World  |


## MailHog (Email interface)
  open url to see recived email.

 ```
  http://localhost:8025/
 ```
